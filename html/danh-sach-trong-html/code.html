Chương trình học "Web developement", gồm các khoá học theo trình tự sau:
  1. HTML
    • Thiết kế cấu trúc của 1 website.
    • Tạo một website đúng chuẩn W3C.
  2. CSS
    • Bố cục và trình bày website 1 cách đẹp mắt.
    • Kỹ thuật tái sử dụng bộ giao diện cho nhiều website khác nhau.
  3. Javascript
    • Xử lý giao diện, dữ liệu phía client
  4. jQuery
    • Sử dụng Javascript đơn giản và hiệu quả bằng thư viện jQuery.
  5. ASP.NET MVC
    • Lập trình xử lý website ở phía server
